##  정렬이란?

* 학교 출석부처럼 순서대로 데이터가 나열되어 있는 것

* 오름차순
* 내림차순





## 정렬 알고리즘의 종류

* 선택 정렬
* 삽입 정렬
* 버블 정렬
* 퀵 정렬





## 선택 정렬

* 개념 : 여러 데이터 중에서 가장 작은 값을 뽑는 작동을 반복하여 값을 정렬

* 최솟값 찾는 방법 

  * 배열의 첫 번째 값을 가장 작은 값으로 지정한다

  * 가장 작은 값을 다음 차례의 값과 비교하여 가장 작은 값을 변경하거나 그대로 두는 방법

  * 마지막 값까지 비교를 마친 후 현재 가장 작은 값으로 지정된 값을 가장 작은 값으로 결정
